## Slide 2: What is Strands?
**Slide Content:**
- Lightweight, code-first framework
- Built for production environments
- Supports multiple model providers
- Focus on simplicity and flexibility

**Speaker Notes:**
Strands is designed with developers in mind. It's a code-first framework that prioritizes simplicity while maintaining the flexibility needed for complex applications. What sets it apart is its production-ready nature and support for various model providers, making it adaptable to your specific needs.

---
## Slide 3: Getting Started
**Slide Content:**
```python
from strands import Agent

# Create an agent
agent = Agent()

# Ask a question
agent("Tell me about agentic AI")
```

**Speaker Notes:**
Getting started with Strands is incredibly straightforward. As you can see, with just a few lines of code, you can create and interact with an AI agent. The framework handles all the complexity behind the scenes, letting you focus on what matters - building your application.
